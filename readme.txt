Pronamic Companies is a basic plugin to build a company directory.

== Description ==

This plugin uses custom post types to add company directory functionality. Pronamic Companies allows you to add, edit and remove companies.

== Installation ==

Upload the Pronamic Companies folder to your wp-content/plugins folder.
